# Skill Packs

Skill Packs are declarative domain modules that define:
- Scenarios (title, description, objectives, difficulty)
- Evaluation rubric (per-skill metrics)
- Prompt templates (system/user/assistant scaffolds)
- Content resources (vocabulary lists, cultural notes)

Format: JSON or YAML. Example below shows a minimal language pack scenario.

See `language-basic.example.json` in this folder.
